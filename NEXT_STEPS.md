## 📋 次のステップ

### 1. テスト用画像の準備（必須）

アプリケーションを動作させるには、画像が必要です。

1. `images`フォルダに画像ファイルを配置
2. ファイル名を `{カテゴリ名}_{番号}.jpg` の形式にする
   - 例: `cat_1.jpg`, `dog_1.jpg`, `bird_1.jpg` など
3. 対応形式: PNG, JPG, JPEG, BMP, GIF

### 2. アプリケーションの起動とテスト

```bash
python main.py
```

#### テスト手順
1. **モード選択**: Blur/Zoom/Hybridのいずれかを選択
2. **画像読み込み**: 「ランダム画像」ボタンをクリック
3. **画像表示確認**: ぼかしやズームが適用されているか確認
4. **回答入力**: 画像の内容を推測して回答を入力
5. **正答判定**: 正解・不正解の判定が正しく動作するか確認
6. **スコア計算**: スコアが正しく計算されるか確認
7. **次の問題**: 「次へ」ボタンで次の画像に切り替わるか確認

### 3. 機能の改善・追加（オプション）

動作確認後、以下の機能追加を検討できます：

#### 優先度: 高
- [ ] **レベル進行の自動化**: 一定時間ごとに自動的にレベルが上がる機能
- [ ] **レベル手動調整**: スライダーやボタンでレベルを手動で変更
- [ ] **正解画像の表示**: 回答後、正解画像を表示する機能

#### 優先度: 中
- [ ] **スコア履歴**: 過去のスコアを記録・表示
- [ ] **カテゴリ選択**: 特定のカテゴリ（動物、乗り物など）のみ出題
- [ ] **難易度設定**: ぼかしの強さやズームの倍率を調整

#### 優先度: 低
- [ ] **効果音**: 正解・不正解時のサウンド
- [ ] **ランキング**: スコアランキング機能
- [ ] **統計表示**: 正答率、平均スコアなどの統計情報
