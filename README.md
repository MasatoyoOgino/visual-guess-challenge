# Visual Guess Challenge

画像当てクイズアプリケーション

## 概要

画像処理技術（ガウシアンぼかし・ズーム処理）を応用した画像当てクイズアプリケーションです。
ぼやけた画像や拡大された画像を徐々に明確にしながら、その内容を推測します。

## 機能

- **Blurモード**: ガウシアンぼかしを段階的に軽減
- **Zoomモード**: 画像の中央部分を段階的に拡大表示
- **Hybridモード**: ぼかしとズームを同時に適用

## 必要な環境

- Python 3.7以上
- PyQt5
- OpenCV
- NumPy

## インストール

```bash
pip install -r requirements.txt
```

## 使い方

### データセットの準備

`images`フォルダに問題用の画像を配置してください。
ファイル名の最初の部分が正解キーワードとして使用されます（例: `cat_1.jpg` → 正解は "cat"）

対応形式: PNG, JPG, JPEG, BMP, GIF

### アプリケーションの起動

```bash
python main.py
```

### 操作方法

1. モードを選択（Blur/Zoom/Hybrid）
2. 「ランダム画像」ボタンでデータセットからランダムに画像を読み込む
   - または「画像読み込み」ボタンで手動で画像を選択
3. 画像を見ながら回答を入力
4. 「回答する」ボタンで回答を提出
5. 「次へ」ボタンで次の問題へ（ランダムに次の画像を読み込み）

## ファイル構成

- `main.py`: メインアプリケーション（PyQt GUI）
- `game_engine.py`: ゲームロジック管理
- `timer_controller.py`: タイマー管理
- `image_processor.py`: 画像処理（ぼかし・ズーム）
- `dataset_loader.py`: データセットローダー（ランダム画像選択）
- `requirements.txt`: 依存関係
- `images/`: 問題用画像フォルダ

## スコア計算

スコア = max(0, 100 - 10×(レベル-1) - 2×経過時間(秒))

早い段階で正解するほど高得点になります。
